Use Postman or curl:
GET all users: GET http://localhost/GOQii/api/
GET single user: GET http://localhost/GOQii/api/1

POST create user:
POST http://localhost/GOQii/api/
Body (raw JSON):
{
  "name": "Alice",
  "email": "alice@example.com",
  "password": "secret123",
  "dob": "1990-01-01"
}

PUT update user: PUT http://localhost/GOQii/api/1
DELETE user: DELETE http://localhost/GOQii/api/1
